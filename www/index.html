Project Darkstar Build Utilities
The sgs-build project contains various utilities that help facilitate building Project Darkstar and its components. This site provides access to the source code repository, issue tracker, and mailing lists for this project. Please see http://www.projectdarkstar.com for more information.
Source code
The publicly accessible Subversion repository contains all of the source code for the sgs-build project.
Issue tracker
Issuezilla is used to track all of the issues with sgs-build including defects, enhancements, and outstanding tasks.
Mailing Lists
Subversion commit notifications and Issue tracker issue change notifications are sent to separate mailing lists for the sgs-build project. For development discussions and general announcements, please join the dev and announce lists for the games-darkstar parent project.
